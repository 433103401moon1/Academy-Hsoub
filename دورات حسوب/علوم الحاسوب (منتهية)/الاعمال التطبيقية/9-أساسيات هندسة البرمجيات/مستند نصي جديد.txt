تثبيت مكتبات الاختبارات في بايثون:
pip install pytest pytest-cov

assert تستخدم في الاختبارات هذه الجملة

نقوم باضافة المكتبات المثبته لدينا في المشروع في ملف حتى يتم تثبيتها تلقائيا خلال الاختبار:
pip freeze > اسم الملف

امر الاختبار:
pytest -v --cov

*تاكد انك داخل ملف العمل عند كتابة الاوامر

استنساخ مستودع داخل جهازنا:
git clone الرابط

جلب اخر التعديلات من المستودع في قيت هب:
git pull

دفع ورفغ اخر التعديلات من جهازنا الى قيت هب:
git push -u origin اسم البرانش

تغيير البرانش او الفرع الذي تعمل به :
git checkout اسم البرانش

لاضافة جميع الملفات المعدله قبل رفعها:
git stage -A

اضافة تعليق:
git commit -m "hello"
